## Troubleshooting

### Not receiving emails?

1. **Check spam folder** - First time might go to spam
2. **Verify Web3Forms dashboard** - Login at https://web3forms.com/
3. **Check browser console** - Look for success/error messages
4. **Wait 2-3 minutes** - Sometimes there's a delay

### Console shows error?

If you see errors in console:
- Check internet connection
- Verify access key is correct
- Try clearing browser cache

### Want to test again?

Clear session storage:
1. Open browser console (F12)
2. Go to Application tab
3. Storage → Session Storage
4. Delete `portfolio_tracked`
5. Refresh page
